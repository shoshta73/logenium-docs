Detailed Description

Header-only logging utility with compile-time format string validation.

This header provides a simple, type-safe logging interface with compile-time format string validation. The logging backend can use either C++20 standard library formatting (std::format/std::println) or fmtlib (fmt::format/fmt::println) based on the LOGENIUM_LOGGING_USE_FMTLIB CMake option.
