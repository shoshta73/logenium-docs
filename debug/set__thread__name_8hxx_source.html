21/**
22 * @brief Set the name of the current thread
23 * @ingroup tracing
24 *
25 * Sets a human-readable name for the current thread that will be visible in
26 * profiling tools. This helps identify threads during performance analysis and
27 * debugging sessions. The thread name is passed directly to Tracy profiler.
28 *
29 * @param name The name to assign to the current thread (null-terminated expected)
30 *
31 * @note The name string should remain valid for the lifetime of the thread or
32 * until another name is set, as Tracy may retain a pointer to it.
33 * @note This function is always available regardless of build configuration,
34 * as Tracy is always linked via TracyClient.
35 *
36 * @code
37 * #include <debug/tracing/set_thread_name.hxx>
38 *
39 * void WorkerThread() {
40 * debug::tracing::SetThreadName("Worker-1");
41 * // Perform work...
42 * }
43 * @endcode
44 */
45void SetThreadName(std::string_view name);
46
47/**
48 * @brief Set the name of the current thread with a group hint
49 * @ingroup tracing
50 *
51 * Sets a human-readable name for the current thread along with a group hint
52 * for organizational purposes in profiling tools. The group hint allows related
53 * threads to be visually grouped together in Tracy's thread view, improving
54 * readability when dealing with many threads.
55 *
56 * @param name The name to assign to the current thread (null-terminated expected)
57 * @param group_hint An integer hint used by the profiler to group related threads
58 *
59 * @note The name string should remain valid for the lifetime of the thread or
60 * until another name is set, as Tracy may retain a pointer to it.
61 * @note The group hint is used by Tracy to organize threads visually. Threads
62 * with the same group hint will be displayed together.
63 * @note This function is always available regardless of build configuration,
64 * as Tracy is always linked via TracyClient.
65 *
66 * @code
67 * #include <debug/tracing/set_thread_name.hxx>
68 *
69 * // Group rendering threads together
70 * void RenderThread(int id) {
71 * debug::tracing::SetThreadName("Renderer-" + std::to_string(id), 1);
72 * // Perform rendering...
73 * }
74 *
75 * // Group worker threads together
76 * void WorkerThread(int id) {
77 * debug::tracing::SetThreadName("Worker-" + std::to_string(id), 2);
78 * // Perform work...
79 * }
80 * @endcode
81 */
82void SetThreadName(std::string_view name, int group_hint);
