Detailed Description

template<typename To>
struct corelib::detail::NullableValueCastFailed< To >

Provides default CastFailed() implementation for nullable types.

This helper struct provides a static CastFailed() method that returns To(nullptr), suitable for nullable types (pointers, smart pointers, etc.). It's used as a base class or building block for cast operations that need to return a null value when a cast fails.

Template Parameters
ToThe target nullable type (must be constructible from nullptr)

Static Methods

  • CastFailed(): Returns To(nullptr) to indicate cast failure
Note
Only suitable for nullable types (pointers, std::unique_ptr, etc.)
Used internally by dyn_cast_or_null() and similar operations

Example:

// For pointer types
int *failed_ptr = NullableValueCastFailed<int*>::CastFailed();
// failed_ptr is nullptr
// For smart pointers
auto failed_unique = NullableValueCastFailed<std::unique_ptr<int>>::CastFailed();
// failed_unique is an empty unique_ptr
