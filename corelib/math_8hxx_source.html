15/**
16 * @defgroup math Mathematical Utilities
17 * @ingroup corelib
18 * @brief Mathematical utilities including vector operations and common math functions.
19 *
20 * The math module provides template-based vector classes for 1D, 2D, 3D, and 4D
21 * mathematical operations. These vectors support various numeric types and provide
22 * common mathematical operations like addition, subtraction, dot products, cross products,
23 * normalization, and length calculations.
24 *
25 * ## Features
26 *
27 * - **Template-based**: Works with any numeric type (float, double, int, etc.)
28 * - **Constexpr support**: Most operations can be evaluated at compile time
29 * - **Multiple naming schemes**: x/y/z/w, r/g/b/a, s/t/p/q for different use cases
30 * - **Type-safe**: Compile-time type checking with debug assertions
31 * - **Zero-cost abstractions**: Optimized for performance with inlining
32 *
33 * ## Components
34 *
35 * ### Vector Classes
36 * - `Vec1<T>`: 1D vector with x/r/s accessors
37 * - `Vec2<T>`: 2D vector with x/y, r/g, s/t accessors
38 * - `Vec3<T>`: 3D vector with x/y/z, r/g/b, s/t/p accessors (includes Cross product)
39 * - `Vec4<T>`: 4D vector with x/y/z/w, r/g/b/a, s/t/p/q accessors
40 *
41 * ### Common Type Aliases
42 * - `dvec1`, `dvec2`, `dvec3`, `dvec4`: Double-precision vectors
43 * - `ivec1`, `ivec2`, `ivec3`, `ivec4`: Integer vectors
44 * - `uvec1`, `uvec2`, `uvec3`, `uvec4`: Unsigned integer vectors
45 *
46 * ## Usage
47 *
48 * Include the convenience header to get all utilities:
49 * @code
50 * #include <corelib/math.hxx>
51 *
52 * using namespace corelib::math;
53 *
54 * // 3D vector operations
55 * Vec3<float> a{1.0f, 2.0f, 3.0f};
56 * Vec3<float> b{4.0f, 5.0f, 6.0f};
57 *
58 * auto sum = a + b;
59 * auto dot = Vec3<float>::Dot(a, b);
60 * auto cross = Vec3<float>::Cross(a, b);
61 * auto normalized = a.Normalized();
62 *
63 * // Type aliases
64 * dvec3 position{1.0, 2.0, 3.0}; // double precision
65 * ivec2 screenPos{100, 200}; // integer coordinates
66 * @endcode
67 *
68 * Or include specific headers:
69 * @code
70 * #include <corelib/math/vector1.hxx>
71 * #include <corelib/math/vector2.hxx>
72 * #include <corelib/math/vector3.hxx>
73 * #include <corelib/math/vector4.hxx>
74 * @endcode
75 */
