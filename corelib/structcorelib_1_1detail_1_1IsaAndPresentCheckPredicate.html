Detailed Description

template<typename... Types>
struct corelib::detail::IsaAndPresentCheckPredicate< Types >

Predicate functor for type checking with presence validation.

IsaAndPresentCheckPredicate wraps isa_and_present<Types...>() in a functor. It checks if a value is present (not null/empty) AND is an instance of any of the specified types.

Template Parameters
TypesThe target types to check for (variadic)
Note
Used internally by public IsaAndPresentPred variable template

Example:

std::vector<Base*> ptrs = ...; // May contain nullptr
IsaAndPresentCheckPredicate<Derived> pred;
auto it = std::find_if(ptrs.begin(), ptrs.end(), pred);
// Finds first non-null element that is a Derived*
