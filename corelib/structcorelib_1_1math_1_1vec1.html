Detailed Description

template<typename T = float>
struct corelib::math::vec1< T >

Template-based 1D vector class.

vec1 represents a one-dimensional vector with a single component accessible through multiple naming schemes (x/r/s) for different use cases.

Template Parameters
TThe component type (defaults to float)
Note
Most operations are constexpr and can be evaluated at compile time
Length() is non-constexpr due to std::abs

Example:

vec1<float> v{5.0f};
v += vec1<float>{2.0f}; // v.x == 7.0f
float len = v.Length(); // len == 7.0f
vec1<float> unit = v.Normalized(); // unit.x == 1.0f
