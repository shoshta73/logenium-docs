Detailed Description

template<typename T>
struct corelib::math::vec4< T >

Template-based 4D vector class.

Vec4 represents a four-dimensional vector with components accessible through multiple naming schemes (x/y/z/w, r/g/b/a, s/t/p/q) for different use cases.

Template Parameters
TThe component type (float, double, int, etc.)
Note
Most operations are constexpr and can be evaluated at compile time
Length() and Normalized() are non-constexpr due to std::sqrt
No Cross() product method (cross product is specific to 3D vectors)

Example:

Vec4<float> a{1.0f, 2.0f, 3.0f, 4.0f};
Vec4<float> b{5.0f, 6.0f, 7.0f, 8.0f};
auto sum = a + b; // (6, 8, 10, 12)
float dot = Vec4<float>::Dot(a, b); // 70
Vec4<float> unit = a.Normalized(); // (0.182, 0.365, 0.547, 0.730)
// RGBA color usage
Vec4<float> color{1.0f, 0.5f, 0.25f, 1.0f};
color.a = 0.5f; // Set alpha to 50%
