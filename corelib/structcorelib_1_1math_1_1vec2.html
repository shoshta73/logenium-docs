Detailed Description

template<typename T>
struct corelib::math::vec2< T >

Template-based 2D vector class.

vec2 represents a two-dimensional vector with components accessible through multiple naming schemes (x/y, r/g, s/t) for different use cases.

Template Parameters
TThe component type (float, double, int, etc.)
Note
Most operations are constexpr and can be evaluated at compile time
Length() and Normalized() are non-constexpr due to std::sqrt

Example:

vec2<float> a{1.0f, 2.0f};
vec2<float> b{3.0f, 4.0f};
auto sum = a + b; // (4, 6)
float dot = vec2<float>::Dot(a, b); // 11
vec2<float> unit = a.Normalized(); // (0.447, 0.894)
