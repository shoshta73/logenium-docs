Detailed Description

template<typename To, typename From, typename Enabler = void>
struct corelib::detail::IsaImpl< To, From, Enabler >

Base template for RTTI-free type checking using classof() pattern.

IsaImpl provides the core type-checking mechanism for the casting system. It delegates to the target type's static classof() method, implementing the LLVM-style RTTI-free type checking pattern. Types must provide a classof() method that returns true if a given pointer is of the target type.

The base template handles the general case by calling To::classof(&Val). A specialization handles the trivial case where From derives from To, which is always true and doesn't require classof().

Template Parameters
ToThe target type to check for
FromThe source type being checked
EnablerSFINAE enabler for specializations

Static Methods

  • Check(const From &Val): Returns true if Val is of type To (via classof())
Note
Requires To to have a static classof(const From*) method
This is the foundation of isa<To>(Val) functionality

Example:

struct Base {
virtual ~Base() = default;
static bool classof(const Base *) { return true; }
};
struct Derived : Base {
static bool classof(const Base *b) {
// Check if b is actually a Derived instance
return; //type check logic
}
};
Derived d;
bool result = IsaImpl<Derived, Base>::Check(d); // Calls Derived::classof(&d)
