template<typename... Types>
struct corelib::detail::IsaCheckPredicate< Types >

Predicate functor for variadic type checking.

IsaCheckPredicate wraps the isa<Types...>() function in a functor suitable for use with standard algorithms. It checks if a value is an instance of any of the specified types.

Template Parameters
TypesThe target types to check for (variadic)
Note
Used internally by public IsaPred variable template

Example:

std::vector<Base*> ptrs = ...;
IsaCheckPredicate<Derived> pred;
auto it = std::find_if(ptrs.begin(), ptrs.end(), pred);
// Finds first element that is a Derived*
