Detailed Description

template<typename To, typename From, typename SimpleFrom>
struct corelib::detail::IsaImplWrap< To, From, SimpleFrom >

Recursive type simplification wrapper for type checking.

IsaImplWrap integrates SimplifyType's recursive type unwrapping with the type-checking infrastructure. It recursively simplifies the From type using SimplifyType until it reaches a "simple" type (one that SimplifyType doesn't further simplify), then delegates to IsaImplConst for the actual check.

This template provides the recursive case, continuing to simplify until From == SimpleFrom (handled by the specialization).

Template Parameters
ToThe target type to check for
FromThe original source type (potentially with qualifiers)
SimpleFromThe simplified version of From (SimplifyType<From>::SimpleType)

Static Methods

  • Check(const From &Val): Recursively simplifies Val and checks type
Note
Recursive template - continues until SimpleFrom is fully simplified
Uses SimplifyType::GetSimplifiedValue() to unwrap values

Example:

// Suppose From = const Base, SimpleFrom = Base
// IsaImplWrap<Derived, const Base, Base>::Check(val)
// → Simplifies const Base to Base
// → Recursively calls IsaImplWrap<Derived, Base, Base>::Check()
// → Reaches specialization, calls IsaImplConst<Derived, Base>::Check()
See also
IsaImplWrap<To, FromTy, FromTy> for the base case (fully simplified)
SimplifyType for the type simplification mechanism
