17/**
18 * @ingroup math
19 * @defgroup math_vectors Vector Classes
20 * @brief Template-based vector classes for mathematical operations.
21 *
22 * This module provides vector classes for 1D, 2D, 3D, and 4D mathematical operations.
23 * All vector classes are template-based and support various numeric types (float, double, int, etc.).
24 *
25 * ## Features
26 *
27 * - **Template-based**: Works with any numeric type
28 * - **Constexpr support**: Most operations can be evaluated at compile time
29 * - **Multiple naming schemes**: Support for different conventions
30 * - Spatial: x, y, z, w
31 * - Color: r, g, b, a
32 * - Texture: s, t, p, q
33 * - **Rich operation set**: Arithmetic, scalar operations, dot/cross products
34 * - **Safety**: Debug assertions for division by zero and zero-length normalization
35 *
36 * ## Vector Classes
37 *
38 * - `Vec1<T>`: 1D vector (x/r/s)
39 * - `Vec2<T>`: 2D vector (x/y, r/g, s/t)
40 * - `Vec3<T>`: 3D vector (x/y/z, r/g/b, s/t/p) with Cross product
41 * - `Vec4<T>`: 4D vector (x/y/z/w, r/g/b/a, s/t/p/q)
42 *
43 * ## Common Type Aliases
44 *
45 * Each vector class provides standard type aliases:
46 * - `dvec1`, `dvec2`, `dvec3`, `dvec4`: Double-precision (double)
47 * - `ivec1`, `ivec2`, `ivec3`, `ivec4`: Signed integer (int)
48 * - `uvec1`, `uvec2`, `uvec3`, `uvec4`: Unsigned integer (unsigned int)
49 *
50 * ## Common Operations
51 *
52 * All vector classes support:
53 * - **Constructors**: Default (zero), uniform value, component-wise
54 * - **Static factories**: Zero(), One()
55 * - **Arithmetic**: operator+, operator-, operator*, operator/ (component-wise and scalar)
56 * - **Compound assignment**: operator+=, operator-=
57 * - **Comparison**: operator==, operator!=
58 * - **Length**: LengthSquared(), Length()
59 * - **Normalization**: Normalize() (mutating), Normalized() (const)
60 * - **Dot product**: Static Dot(a, b) method
61 * - **Cross product**: Static Cross(a, b) method (Vec3 only)
62 *
63 * ## Usage Examples
64 *
65 * @code
66 * using namespace corelib::math;
67 *
68 * // Construction
69 * Vec3<float> zero; // (0, 0, 0)
70 * Vec3<float> ones{1.0f}; // (1, 1, 1)
71 * Vec3<float> pos{1.0f, 2.0f, 3.0f}; // (1, 2, 3)
72 *
73 * // Arithmetic operations
74 * Vec3<float> a{1.0f, 2.0f, 3.0f};
75 * Vec3<float> b{4.0f, 5.0f, 6.0f};
76 * auto sum = a + b; // (5, 7, 9)
77 * auto scaled = a * 2.0f; // (2, 4, 6)
78 *
79 * // Dot and cross products
80 * float dot = Vec3<float>::Dot(a, b);
81 * Vec3<float> cross = Vec3<float>::Cross(a, b);
82 *
83 * // Normalization
84 * Vec3<float> dir{3.0f, 0.0f, 4.0f};
85 * float len = dir.Length(); // 5.0
86 * auto unit = dir.Normalized(); // (0.6, 0, 0.8)
87 * dir.Normalize(); // Mutates dir to (0.6, 0, 0.8)
88 *
89 * // Color naming scheme
90 * Vec4<float> color{1.0f, 0.5f, 0.25f, 1.0f};
91 * color.r = 0.8f; // Modify red component
92 * color.a = 0.5f; // Modify alpha component
93 *
94 * // Type aliases
95 * dvec3 position{1.0, 2.0, 3.0}; // double precision
96 * ivec2 screenPos{800, 600}; // integer coordinates
97 * uvec4 rgba{255, 128, 64, 255}; // unsigned color
98 * @endcode
99 *
100 * @see Vec1
101 * @see Vec2
102 * @see Vec3
103 * @see Vec4
104 */
