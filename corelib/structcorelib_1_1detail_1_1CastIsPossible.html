Detailed Description

template<typename To, typename From, typename Enable>
struct corelib::detail::CastIsPossible< To, From, Enable >

Base template for checking if a cast is possible.

CastIsPossible determines whether a cast from From to To can succeed by delegating to the IsaImplWrap infrastructure. This integrates type simplification with the classof()-based type checking.

The base template handles general types by simplifying and checking. Specializations handle:

  • std::optional (unwraps and checks the contained value)
  • Trivial upcasts (From derives from To, always possible)
