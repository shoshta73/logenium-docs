16/**
17 * @defgroup utility Utility Module
18 * @brief Core utility functions and RAII wrappers
19 *
20 * The utility module provides essential tools for C++ development including:
21 * - **Type introspection**: Compile-time type name extraction using compiler intrinsics
22 * - **RAII utilities**: Automatic resource management and scope-exit execution
23 * - **Generic wrappers**: Adapters for managing C-style resources with custom cleanup
24 *
25 * All utilities are designed to be zero-cost abstractions that leverage C++23 features
26 * like constexpr evaluation, perfect forwarding, and move semantics.
27 *
28 * ## Components
29 *
30 * ### Type Introspection
31 * - `type_name<T>()`: Extract type names at compile time for debugging and diagnostics
32 *
33 * ### RAII Utilities
34 * - `Defer`: Execute arbitrary code at scope exit (like Go's defer or D's scope(exit))
35 * - `AutoRelease`: Manage resources with custom releasers (file descriptors, handles, etc.)
36 *
37 * ## Usage
38 *
39 * Include the convenience header to get all utilities:
40 * @code
41 * #include <corelib/utility.hxx>
42 * @endcode
43 *
44 * Or include specific headers:
45 * @code
46 * #include <corelib/utility/type_name.hxx>
47 * #include <corelib/utility/defer.hxx>
48 * #include <corelib/utility/auto_release.hxx>
49 * @endcode
50 */
