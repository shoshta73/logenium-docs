Detailed Description

template<typename U>
struct corelib::detail::CastIfPresentFunc< U >

Functor for conditional cast operations.

CastIfPresentFunc wraps cast_if_present in a functor for use with algorithms. Uses perfect forwarding to preserve value categories.

Template Parameters
UThe target type for the cast
Note
Used internally by public CastIfPresentTo variable template

Example:

std::vector<std::optional<Base>> opts = ...;
std::vector<Derived> deriveds;
std::transform(opts.begin(), opts.end(),
std::back_inserter(deriveds),
CastIfPresentFunc<Derived>{});
// Casts only when optional has value
