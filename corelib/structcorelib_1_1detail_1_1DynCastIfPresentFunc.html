Detailed Description

template<typename U>
struct corelib::detail::DynCastIfPresentFunc< U >

Functor for conditional dynamic cast operations.

DynCastIfPresentFunc wraps dyn_cast_if_present in a functor for use with algorithms. Uses perfect forwarding to preserve value categories.

Template Parameters
UThe target type for the cast
Note
Used internally by public DynCastIfPresentTo variable template

Example:

std::vector<std::optional<Base*>> opts = ...;
std::vector<Derived*> deriveds;
std::transform(opts.begin(), opts.end(),
std::back_inserter(deriveds),
DynCastIfPresentFunc<Derived*>{});
// Casts only when optional has value, returns nullptr on failure
